El browser presentará un diálogo donde el usuario ingresa una palabra o una frase.
Al aceptar, deberá presentarse un mensaje indicando si el texto ingresado es un palíndromo o no.

Un palíndromo es un texto que se puede leer igual al derecho o al revés. Es decir capicúa.

Completá el código fuente.
Para volver a probar, usá F5.
Para ver la consola, usá F12.
